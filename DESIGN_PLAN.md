## 2. 2026 UI/UX 디자인 트렌드 적용

### 핵심 트렌드
1. **Bento Grid Layout** - 비대칭 카드 기반 모듈형 레이아웃
2. **Glassmorphism** - 반투명 배경, 블러 효과, 부드러운 그림자
3. **Pastel & Soft Tones** - 차분하고 현대적인 파스텔 컬러
4. **Minimalist Clean Design** - 불필요한 요소 제거, 화이트스페이스 활용
5. **Rounded Typography & Icons** - 친근한 둥근 폰트와 아이콘

### 새 컬러 팔레트 (밝은 계열)
```css
:root {
    /* Primary - Warm Amber/Gold (유지하되 더 밝게) */
    --primary: #f59e0b;
    --primary-light: #fef3c7;
    --primary-dark: #d97706;

    /* Background - Clean White & Soft Gray */
    --bg-base: #ffffff;
    --bg-soft: #fafaf9;
    --bg-muted: #f5f5f4;
    --bg-card: rgba(255, 255, 255, 0.8);

    /* Text */
    --text-primary: #1c1917;
    --text-secondary: #57534e;
    --text-muted: #a8a29e;

    /* Accent Colors */
    --success: #22c55e;
    --warning: #f59e0b;
    --error: #ef4444;
    --info: #3b82f6;

    /* Glass Effect */
    --glass-bg: rgba(255, 255, 255, 0.7);
    --glass-border: rgba(255, 255, 255, 0.3);
    --glass-shadow: 0 8px 32px rgba(0, 0, 0, 0.08);

    /* Border & Radius */
    --border: #e7e5e4;
    --radius-sm: 8px;
    --radius-md: 12px;
    --radius-lg: 16px;
    --radius-xl: 24px;
}
```

## 6. 반응형 브레이크포인트

```css
/* Mobile First */
@media (min-width: 640px)  { /* sm */ }
@media (min-width: 768px)  { /* md */ }
@media (min-width: 1024px) { /* lg */ }
@media (min-width: 1280px) { /* xl */ }
```

### 레이아웃 변화
- **Mobile (< 768px)**: 사이드바 → 하단 네비게이션
- **Tablet (768px-1024px)**: 접이식 사이드바
- **Desktop (> 1024px)**: 고정 사이드바

## 7. 애니메이션 & 인터랙션

### 기본 트랜지션
```css
.transition-base {
    transition: all 0.2s ease-in-out;
}

.transition-slow {
    transition: all 0.3s ease-in-out;
}
```

### 호버 효과
- **버튼**: 약간 위로 이동 + 그림자 확대
- **카드**: 그림자 확대 + 테두리 강조
- **링크**: 색상 변화 + 밑줄 애니메이션

### 로딩 상태
- **스켈레톤**: 회색 블록 펄스 애니메이션
- **스피너**: 회전하는 원형 인디케이터
