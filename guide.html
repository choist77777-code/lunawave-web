1 Getting Started

System Requirements

Minimum

  • Windows 10 (64-bit)
  • CPU: Intel i5 / AMD Ryzen 5
  • RAM: 8 GB
  • Storage: 2 GB
  • GPU: Integrated graphics

Recommended

  • Windows 10/11 (64-bit)
  • CPU: Intel i7 / AMD Ryzen 7
  • RAM: 16 GB
  • Storage: 5 GB+ SSD
  • GPU: NVIDIA GTX 1060+ (NVENC)
NVIDIA GPU(GTX 1060 이상)가 있으면 하드웨어 인코딩(NVENC)으로 렌더링 속도가 5~10배 빨라집니다.

Installation

  1. lunawaveapp.com/download에서 최신 버전 다운로드
  2. 다운로드한 LUNA WAVE Setup x.x.x.exe 실행
  3. 설치 완료 후 앱이 자동 실행됩니다
  4. 계정 로그인 또는 회원가입 후 사용 시작
앱은 자동 업데이트를 지원합니다. 새 버전이 있으면 시작 시 자동으로 업데이트됩니다.

2 Interface

LunaWave의 메인 화면은 상단 탭으로 구성되어 있습니다. 각 탭에서 AI 가사 생성, 곡 관리, 자동 러너, 비디오 편집 등 모든 기능에 접근할 수 있습니다.

LunaWave Main Interface - Tab Navigation

Tab Structure

  • Generate - AI 기반 가사 자동 생성
  • Songs - 생성된 곡 목록 관리
  • Runner - Suno AI 자동 입력 러너
  • Suno - Suno 곡 다운로드
  • YouTube - YouTube 메타데이터 생성
  • Video - Engine2 비디오 빌더 (별도 창)
  • Analytics - 채널 분석
  • Settings - 앱 설정
테마는 Light / Dark 모드를 지원합니다. Settings에서 변경하거나, 상단 바의 테마 버튼으로 전환하세요.

3 Engine2 Video Builder

Engine2는 음악 비주얼라이저 영상을 제작하는 핵심 모듈입니다. 오디오 파일과 이미지를 업로드하고, 비주얼라이저, 텍스트, 자막, 컷아웃 효과를 적용하여 전문적인 뮤직 비디오를 만들 수 있습니다.

Engine2 Full Screen Layout

Layout Overview

  1. Left Panel: File, Visualizer, Text/Font, Subtitle, Cutout, Output, Render section (accordion)
  2. Center Area: Preview canvas, meta info, status, log
  3. Each accordion header toggles the section open/close
Left panel sections are arranged in workflow order from top to bottom: File → Visualizer → Text → Cutout → Output → Render

3.1 Top Bar

Engine2 Top Bar
  1. LunaWave Logo: Click to return to the main page
  2. GPU Badge: Shows detected GPU name
  3. Luna Balance: Shows current Luna points balance
  4. Theme Button: Toggle between Light/Dark mode

3.2 File Upload

File Upload Section
  1. Audio files: Select audio files (MP3, WAV, FLAC, etc.). Multiple tracks auto-concatenate
  2. URL list: Upload a text file with YouTube URLs for batch download
  3. Image Mode: "1 Image" for single background, "2 Images" for separate person + background
  4. Image (required): Select background image. PNG, JPG, WEBP supported
Text Behind Person or Cutout requires an image with a person in it.
Both audio and image must be uploaded before rendering is possible.

3.3 Visualizer

Visualizer Settings
  1. Template/Shape: Choose from Bars, Line Wave, Dots
  2. Preset Grid: 12 presets (Classic, Mirror, Rounded, etc.) - click to apply
  3. Color: Set visualizer color. Check Gradient for 2-color gradient
  4. Direction / Bar count / Align / Sensitivity: Fine-tune direction, bar count, alignment, and sensitivity
Use Quick Position buttons to move the visualizer in one click.
Style Presets like "Neon Night", "Lo-fi Dream" apply complete visual themes.

3.4 Text Edit

Text Edit Panel
  1. Text items: Add (+), delete, duplicate, copy/paste styles for multiple text layers
  2. Text input: Enter text. Use "Line break" button for new lines
  3. X / Y: Position in pixels. Also adjustable by dragging on preview
  4. Align / Font size / Opacity / Line spacing: Alignment, size, transparency, line spacing
Click text directly on the preview to select, drag to move.
Ctrl+Shift+C copies style, Ctrl+Shift+V pastes style to another text item.

3.5 Font

Font Selection Panel
  1. Font dropdown: Browse and select from system fonts
  2. Preview: Real-time sample text preview with selected font
  3. Favorites: Add fonts to favorites for quick access
  4. Select from file: Load TTF/OTF font files directly
For Korean fonts, use malgun (Malgun Gothic), gulim, or load a TTF file directly.

3.6 Color / Gradient

Color and Gradient Settings
  1. Font color: Set the base text color
  2. Enable font gradient: Check to apply vertical gradient
  3. Preset chips: Sunset, Ocean, Neon, Gold, etc. - click to apply
  4. Color 1 / 2 / 3: Manually set top, middle, bottom gradient colors
"Sync font color with Visualizer" automatically matches text and visualizer colors.

3.7 Subtitle / Lyrics

Subtitle / Lyrics Settings
  1. SRT file upload → Overlay subtitles on video
  2. Font size, Highlight style: None, Current Line, Karaoke Fill
  3. Offset slider: Fine-tune timing from -5s to +5s
If sync is off, adjust the Offset slider.
Only SRT format is supported for subtitle files.

3.8 Cutout / Effects

Cutout / Effects Panel
  1. Auto: AI automatic person extraction. Enable Fast Mode for quicker processing
  2. Slow: Higher quality cutout processing
  3. View: Preview the cutout result
  4. Text behind person: Place text behind the extracted person
"Ultra (BiRefNet)" provides the most precise cutout results.
Cutout is only available after uploading an image.

3.9 Output Settings

Output Settings Panel
  1. Aspect: 16:9 (YouTube) or 9:16 (Shorts)
  2. Resolution: 1080p or 720p
  3. Advanced: Encoder, quality preset, CRF, and other advanced settings
  4. Presets: Save setting presets, Export/Import for sharing
Auto encoder = GPU auto-detection. Best performance with NVIDIA.
Use the Storage panel to check disk usage and Clear All to clean up.

3.10 Render

Render Execution Panel
  1. Render Video (20 Luna): Start final FFmpeg rendering
  2. Queue Manager: Process multiple videos in sequence
  3. Shorts: Dedicated 9:16 vertical video page
  4. Download timestamps: Download YouTube chapter timestamps
Do not close the browser during rendering. Progress will be lost.

3.11 Preview

Preview Canvas and Controls
  1. Preview canvas: Real-time display of all applied settings
  2. Guide button: Toggle center line / rule-of-thirds guidelines
  3. Preview 3 Seconds: Generate a 3-second preview video
  4. Clear Preview: Reset the preview
Check image size, resolution, and font info in the meta information area.
Preview is for reference. Final FFmpeg output may differ slightly.

Song Management Tab

6 Auto Input Runner

Suno AI에 곡 정보를 자동으로 입력하는 자동화 도구입니다.

Auto Input Runner Tab

Calibration

Runner를 처음 사용하거나 Suno 레이아웃이 변경된 경우 캘리브레이션이 필요합니다.

  1. Calibration 버튼 클릭
  2. Suno 페이지에서 Title 입력 필드 클릭
  3. Prompt 입력 필드 클릭
  4. Lyrics 입력 필드 클릭
  5. Create 버튼 위치 클릭
Runner Calibration Detail

Queue Management

  • Add Song: Add songs to the runner queue
  • Edit / Remove: Modify or delete queue items
  • Clear All: Remove all items from queue
  • Import JSON: Bulk import from JSON file
  • Sync: Sync with Songs tab

Execution

  1. Click Start Auto Input Runner
  2. Press F8 to begin automated input
  3. Press ESC to stop at any time
Runner Auto Mode in Action
Batch limit: 30-second pause every 10 songs to avoid rate limiting.

7 Suno Downloader

Suno AI에서 생성한 곡을 자동으로 다운로드합니다.

Auto Connect

  1. Click the Auto Connect button
  2. Browser opens → Log in to Suno
  3. Cookie is automatically collected

Download

Connection issues? Make sure you are logged into Suno, then try reconnecting.

12 Troubleshooting

Issue Solution
App does not start Check if localhost:3000 is in use by another app. Check firewall settings.
Rendering is slow Ensure NVIDIA GPU is detected. Update GPU drivers to the latest version.
Performance drops over time Check CPU temperature and ventilation. Close unnecessary background apps.
Cutout not working Verify the Cutout server is running. Restart the app if needed.
Canvas differs from FFmpeg output Use "Accurate Preview" or "Preview 3 Seconds" to verify.
Suno connection failed Check login status on Suno. Try reconnecting via Auto Connect.
Visualizer not visible Make sure audio is uploaded and a visualizer preset is selected.
