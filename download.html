시스템 요구사항

  • Windows 10 (64-bit) 이상
  • RAM 8GB 이상 (16GB 권장)
  • 저장공간 2GB 이상
  • 인터넷 연결 필요
  • CPU: Intel Core i5 / AMD Ryzen 5 이상
  • GPU: DirectX 11 지원 (렌더링 가속)

설치 방법

1

다운로드

위의 다운로드 버튼을 클릭하여 설치 파일을 받습니다.

2

설치 실행

다운로드된 LUNA-WAVE-Setup-6.0.10.exe를 실행합니다. Windows 보안 경고가 뜨면 "추가 정보" → "실행"을 클릭합니다.

3

설치 완료

설치 마법사의 안내에 따라 설치를 완료합니다. 기본 설정을 권장합니다.

4

로그인

앱을 실행하고 LunaWave 계정으로 로그인합니다. 계정이 없다면 먼저 회원가입하세요.

설치 문제 해결

  • 설치가 안 되면 바이러스 백신을 잠시 비활성화하고 다시 시도해보세요.
  • 관리자 권한으로 실행해보세요 (우클릭 → 관리자 권한으로 실행).
  • .NET Framework 4.8 이상이 필요합니다. 없으면 자동으로 설치됩니다.
  • 문제가 계속되면 고객센터로 문의해주세요.
