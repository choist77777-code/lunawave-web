System requirements

  • Windows 10 (64-bit) or macOS 12+
  • 8GB RAM minimum (16GB recommended)
  • 2GB free disk space
  • Internet connection required
  • CPU: Intel Core i5 / AMD Ryzen 5 / Apple M1 or better
  • GPU: DirectX 11 / Metal (hardware render acceleration)

How to install

1

Download

Click the download button above for your OS.

2

Install (Windows)

Run LUNA.WAVE.Setup.6.1.23.exe. If SmartScreen warns, click "More info" → "Run anyway".

2

Install (macOS)

Open LUNA.WAVE-6.1.23.dmg. Double-click 🚀 Install.command to auto-install. You may need to click "Open" once to confirm.

3

Launch

Windows: follow the installer. macOS: the app is auto-copied to Applications and launched.

4

Log in

Launch LunaWave and sign in. Don't have an account? Sign up — it's free.

Installation troubleshooting

  • If installation fails, try pausing your antivirus and run again.
  • Try running as administrator (right-click → Run as administrator).
  • .NET Framework 4.8+ is required (auto-installed if missing).
  • macOS: If Gatekeeper blocks the app, the included 🚀 Install.command handles it automatically.
